# viatctl
CLI tool for Aviator

# Purpose
Management tool to control Aviator service through CLI environment

# Usage (Developer)
## 1. clone Github Repository
```bash
$ git clone https://github.com/cloud-club/viatctl.git
```

## 2. Build viatctl
```bash
$ go build
```

# Environment
- GO : 1.21.1
